# NEX
## An Autocomplete Experiment

### To run it locally
Make sure you have node and npm installed!
If you don't, try:
[nodejs.org](http://nodejs.org)

```
$ git clone https://github.com/sheetaluk/nex
$ cd nex
$ npm start
```
Now, navigate to:
[localhost:8000](http://localhost:8000/app/index.html)
